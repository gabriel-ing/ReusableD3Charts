Reusable plots

These are plots created with D3.js which are designed to be as Reusable as possible. As such, the inputs into the plots are separated from the plotting logic. The plots can also be replotted with different data, with some basic transition animations. There is also a specified function to download a static SVG image of the plot, which can be used in other programs including as raw images in a PowerPoint or Word document, or as an editable SVG in Illustrator or other graphic design programs.
